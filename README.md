# Aura Ciclo - Registro Menstrual

A modern menstrual cycle tracking application with AI-powered insights, built with React and Vite as a Progressive Web App (PWA).

## Features

- Menstrual cycle tracking and prediction
- Mood and symptom logging
- AI-generated phase insights
- Calendar view
- Offline support via PWA

## Prerequisites

- Node.js (version 18 or higher recommended)

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd aura-ciclo---registro-menstrual
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Setup

1. Obtain a Gemini API key from [Google AI Studio](https://ai.google.dev/).

2. Create a `.env.local` file in the root directory and add your API key:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

## Running the App

To start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173` (default Vite port).

## Building for Production

To build the app for production:
```bash
npm run build
```

To preview the production build:
```bash
npm run preview
```
